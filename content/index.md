用于上传我的数学笔记

### 进入终端：
cd D:\MyMathBlog\quartz\quartz\
(tap可达)
### 上传文件所用代码：

- `git add .`

- `git commit -m "这里写备注"`

- `git push`

### 预览：

终端输入：
npx quartz build --serve

访问：
localhost:8080

ctrl + c 结束预览

### 意外：
如果因为没有留言而进入Vim,

- 按 `i` 进入编辑模式
    
- 写你的留言
    
- 按 `Esc` 退出编辑模式
    
- 输入 `:wq` 保存并退出
